Die Suchmaske

In der Suchmaske befindet sich als erstes eine Zeile mit allen Buchstaben und ein Stern.
Klicken Sie einen der Buchstaben an, so wird sofort eine Suche nach allen Firmen mit diesem Anfangsbuchstaben gestartet. Der Stern sucht nach Firmen die nicht mit einem Buchstaben beginnen, z.B. einer Zahl oder einem Sonderzeichen.

Darunter befinden sich für sehr viele Felder der Datenbank je ein Eingabefeld. Geben Sie dort Ihren Suchbegriff ein. Je mehr Felder Sie ausfüllen, desto mehr wird das Suchergebnis eingegränzt. So genannte Jokerzeichen sind erlaubt:
? oder _ stehen für ein beliebiges Zeichen
* oder % für beliebig viele Zeichen

meier findet Meier aber nicht Meierei
meier* findet Meier und Meierei
m?ier findet Meier und auch Maier ....
*meier findet Hintermeier

  • Jedes Suchwort kann intern automatisch mit einem Stern ergänzt werden. Dazu muß das Häkchen "Unscharf Suche" aktiviert sein.
  • Wollen Sie Ihren Begriff beliebig in einem Feld suchen, muß ein Stern vorangestellt werden. Dieser Prefix kann auch hinterlegt werden und muß dann nur weiter unten aktiviert werden.
  • Soll die Suche nicht in abweichenden Lieferanschriften durchgeführt werden, so deaktivieren sie die entsprechende Checkbox.
  • Auch kann die Suche auf Adressen beschränkt werden, die der angemeldete Benutzer angelegt hat.

Starten Sie die Suche mit dem Knopf suche. Mit löschen können Sie die Maske zurücksetzen.

Suchergebnis

Wurde kein Treffer in der Datenbank erziehlt, so erscheint die Suchmaske sofort wieder mit den eingegebenen Werten und einer Fehlermeldung.

Werden mehrere Treffer erziehlt, so werden diese auf einer neuen Seite als Liste ausgegeben.
Mit einem Klick auf einen Eintrag wird der Datensatz sofort in der Stammdatenmaske angezeigt.
Es kann mit der Treffermenge auch eine [Massenaktion] gestartet werden.

Bei nur einem Treffer wird sofort in die Stammdatenmaske gewechselt.

Reportgenerator
Die Suche kann als Report in einer CSV-Datei gespeichert werden.

Es öffnet sich zunächst ein neues Fenster mit drei Auswahlboxen. Die erste Box stellt die Datenbanfelder der Kunden-/Lieferantenadresse bereit, die zweite Box die Felder der abweichenden Anschriften. In der letzten Box stehen die Felder der Ansprechpartner bereit.
Wählen Sie die benötigten Felder in den Auswahlboxen aus. Für Mehrfachselektionen müssen Sie die Taste strg gedrückt halten.
Mit dem Knopf übernehmen starten Sie den Generator. Das Fenster schließt sich wieder und eine leere Suchmaske wird wieder angezeigt. Unterhalb der Aktionsknöpfe wird Ihnen nun ein Link angeboten, mit dem Sie die Daten lokal speichern können.

Es kann vorkommen, dass eine Firma mehrfach in der Datei aufgenommen wurde. Das liegt daran, dass es dann mehrere Ansprechpartner oder abw. Adressen gibt.

Massenaktionen
Serienbriefe, Serien E-Mail und Etiketten können mit der Ergebnismenge erstellt werden. Dafür werden entsprechende Aktionsknöpfe bereitgestellt. Etiketten sind voreingestellt.

Etiketten

Wählen Sie zunächst das passende Etikett aus. Im folgenden Eingabefeld können Sie noch einen freien Text eingeben. Zuletzt bestimmen Sie die Position, bei der mit der Etikettenerstellung begonnen werden soll. z.B. wenn bereits zwei Etiketten fehlen.
Klicken Sie auf erzeugen, es wird Ihnen ein PDF-Dokument für den Ausdruck bereitgestellt.


Serienbrief
Die einizgen freien Felder für den Serienbrief sind DATE, SUBJECT und BODY.
Geben Sie in den Felder Ihren gewünschten Text ein. Wählen Sie noch die passende Vorlage auf Ihren lokalen Laufwerken und klicken Sie auf erzeugen.
Es öffnet sich ein neues Fenster. Dieses bitte erst nach Aufforderung schließen.

Serien E-Mail
Beim angemeldete Benutzer muß eine E-Mail Adresse eingestellt sein. Mit dieser werden die E-Mails verschickt.

Es öffnet sich die E-Mail Maske. Sie können einen weiteren Empfänger eingeben. Geben Sie nun einen eindeutigen Betreff ein und den Text der Nachricht. Eine Datei können sie an die Mail anhängen.
Klicken Sie auf senden. Ein Fenster öffnet sich. Bitte erst nach Aufforderung schließen.
