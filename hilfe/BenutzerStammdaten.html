BenutzerStammdaten

Hier kann der Benutzer seine "Stammdaten" editieren. Die meisten Felder sind wohl selbsterklärend.

Die Felder im einzelnen:


-Kennwort: Hier kann das Passwort zu diesem Account verändert werden

-Mailserver: Name des Mailservers, auf dem der Benutzeraccount liegt

-Postfach: Name des Postfaches, meist der Useraccount auf dem Mailserver

-Kennwort: Das Passwort für das Mailfach

-Intervall: wie oft soll nach neuen Mails gesehen werden, betr. Wiedervorlage

-PreSerch: wird automatisch bei der Schnellsuche vorrangestellt, z.B. "%"

-Vertreter: welcher Benutzer übernimmt meine Vertretung, nur wichtig mit dem Telefonapplet. Siehe auch geschäftl.

-eMail: diese Mailadresse wird beim versenden benutzt

-geschäftl.: Telefonnummer unter der der Benutzer erreichbar ist. Wichtig mit Telefonapplet.

-Etikett: Das voreingestellte Etikett für Ausdrucke

-Sprache: Es kann auf verschiedene Sprachen umgestellt werden - erst Ansatzweise implementiert

Ist der Benutzer Gruppen zugeordnet, werden diese unter der Mailunterschrift angezeigt.


Mit dem Button "sichern" werden die geänderten Einstellungen gespeichert.
